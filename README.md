## React Fifteen

## Installation
* `git clone <repository-url>` this repository
* `cd fifteen-react`
* `npm i`

## Running / Development
* `npm start`
* Visit your app at [http://127.0.0.1:3000/](http://127.0.0.1:3000).

## Building / Production
* `npm run build`
* Builds the app for production to the build folder.
